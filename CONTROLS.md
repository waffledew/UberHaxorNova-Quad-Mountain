# Important controls

## Not Enough Items

- `O` — show or hide the NEI interface.
- `R` while hovering an item — show its recipe.
- `U` while hovering an item — show what it can be used to make.
- Type in the search field — filter the item list.

## Inventory Tweaks

- `K` — sort the current inventory. This was moved away from `R` so it does not conflict with NEI recipes.
- Middle-click — sort a supported inventory or chest.

## General

- `E` — inventory.
- `Z` — OptiFine zoom, depending on the imported key configuration.

Individual legacy mods may expose additional controls in Minecraft's Controls menu or their files under `.minecraft/config`.
