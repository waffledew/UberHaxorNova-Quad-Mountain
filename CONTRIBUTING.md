# Contributing

Thank you for helping preserve this early modded-Minecraft setup.

## Good contributions

- Archived source links and author information.
- License or redistribution evidence.
- Reproducible Minecraft 1.1 compatibility findings.
- Documentation corrections.
- Crash reports that include the MultiMC console log and exact reproduction steps.
- Source-level compatibility patches that can be reviewed without redistributing an unauthorized original binary.

## Please do not submit

- Minecraft game jars or Mojang assets.
- Account files, access tokens, or personal launcher data.
- Third-party mod binaries without documented redistribution permission.
- Decompiled source presented as original work.
- Modern mod versions merely renamed to look compatible with Minecraft 1.1.

## Technical reports

Include the Minecraft version, Java version, relevant mod versions, what you expected, what happened, and the smallest set of steps that reproduces the issue.

## Attribution reports

Link to primary or archived evidence whenever possible. Clearly separate confirmed facts from recollection or inference.
