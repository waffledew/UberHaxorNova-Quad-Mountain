# Installing in MultiMC

## Import the finished ZIP

1. Install MultiMC or a compatible fork.
2. Make sure a Java 8 runtime is installed. Minecraft 1.1 is most reliable with Java 8.
3. In MultiMC, click **Add Instance**.
4. Choose **Import from ZIP**.
5. Browse to the Quad Mountain Survival export ZIP.
6. Finish the import and select the new instance.
7. Open **Edit Instance → Settings → Java** and select your Java 8 installation if MultiMC did not choose it automatically.
8. Launch once and allow MultiMC to download the official Minecraft and library files.

## Expected first launch

The game should reach the main menu. Open a world and press `E` to confirm that Not Enough Items appears around the inventory.

The included **Quad Island Survival** world should appear in the Singleplayer world list. Its original `Quad Island Survival.rar` archive is also preserved inside `.minecraft/saves`; Minecraft ignores the archive and loads the extracted folder.

If the NEI item panel is hidden, press `O` once while the inventory is open.

## Memory

The pack normally needs no more than 1–3 GB of memory. Allocating excessive memory does not improve Minecraft 1.1.

## Do not add modern versions

Every mod in this instance is tied to Minecraft 1.1. Mods labeled 1.1.2, 1.2.5, 1.4.6, or newer usually refer to a different Minecraft release and should not be substituted without a deliberate backport.
